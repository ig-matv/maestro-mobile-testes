# Automação Mobile com Maestro – AnkiDroid

## Sobre

Este projeto contém uma automação mobile desenvolvida com Maestro no aplicativo AnkiDroid.

O objetivo é simular um fluxo real de usuário, desde a criação de um deck até a interação com o sistema de estudo dos cards.

---

## Cenário Automatizado

O fluxo executado pelo teste é:

- Abrir o aplicativo
- Criar um novo deck
- Definir o nome do deck
- Criar um card 
- Salvar o card
- Validar a criação do deck 
- Acessar o deck criado
- Executar o fluxo de estudo 

---

## Tecnologias Utilizadas

- Maestro
- Android

# English Version

# Mobile Automation with Maestro – AnkiDroid

## About
This project contains a **mobile automation** developed with Maestro on the AnkiDroid app.  

The goal is to simulate a real user flow, from creating a deck to interacting with the card study system.

---

## Automated Scenario
The test executes the following flow:

1. Open the app  
2. Create a new deck  
3. Set the deck name  
4. Create a card  
5. Save the card  
6. Validate the deck creation  
7. Access the created deck  
8. Execute the study flow
