# Automação Mobile com Maestro – AnkiDroid

## Sobre

Este projeto contém uma automação mobile desenvolvida com Maestro no aplicativo AnkiDroid.

O objetivo é simular um fluxo real de usuário, desde a criação de um deck até a interação com o sistema de estudo dos cards.

---

## Cenário Automatizado

O fluxo executado pelo teste é:

- Abrir o aplicativo
- Criar um novo deck
- Definir o nome do deck
- Criar um card 
- Salvar o card
- Validar a criação do deck 
- Acessar o deck criado
- Executar o fluxo de estudo 

---

## Tecnologias Utilizadas

- Maestro
- Android
